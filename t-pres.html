Lets create a presentation.

Context

Use Impress in LibreOffice to create effective multimedia presentations. Impress makes editing extremely easy and allows you to be very creative.

Pre-requisites

  • LibreOffice 4.3

Steps

  1. Open LibreOffice 4.3
  2. Select "Impress Presentation" to create a new presentation.
  3. Fill in the slides with your material

    To insert a new slide select "Insert" then "Slide"

  4. Use preloaded formats in Impress to customize your presentation.
  5. When completed, under file select "Save As" and select a destination.
