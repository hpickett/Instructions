Lets create a chart.

Context

LibreOffice embeds the ability to create and embed charts. Charts can be defined not just by defining its data, its style, color and size can be customized in a broad variety of ways: pie charts, square and round colums, trend graphs, dots, 2D and 3D charts...

Pre-requisites

  • LibreOffice 4.3

Steps

  1. Open LibreOffice 4.3
  2. Select "Calc Spreadsheet" to create a new spreadsheet.
  3. Create a data ranges for the chart within the spread sheet
  4. When ready to make chart highlight all data and select "Insert" at the top menu bar.
  5. Select the type of chart you would like to make.
  6. When completed, under file select "Save As" and select a destination.
